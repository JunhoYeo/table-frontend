## 사용법

Pillow 모듈이 설치되어 있는 Python 3 환경이여야 합니다.

![](images/img5.png)

`python app.py`로 웹서버를 시작하세요.

`mario.png` 파일을 다른 이미지로 바꾸고 `http://localhost:5000`에 접속해서 HTML 코드를 확인하시면 됩니다.
